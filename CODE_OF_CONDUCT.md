# Code of Conduct

## Our Pledge

We aim to make participation in this project respectful, welcoming, and useful for everyone.

## Expected Behavior

- Use welcoming and inclusive language.
- Be respectful of different viewpoints.
- Accept constructive feedback.
- Focus on what is best for the community.
- Follow ethical research and data-use practices.

## Unacceptable Behavior

- Harassment or abusive language
- Personal attacks
- Sharing private or restricted data
- Promoting unsafe medical advice
- Adding spam or unrelated resources

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and related community channels.
