# Contributing to Awesome Breast Cancer

Thank you for your interest in contributing.

You can contribute by adding:

- New datasets
- Recent papers
- Open-source code
- Benchmark results
- Survey papers
- Useful tools
- Tutorials
- Reproducible experiments

---

## How to Add a Dataset

Please use this format:

```text
### Dataset Name

- Full Name:
- Modality:
- Tasks:
- Classes/Labels:
- Number of Samples:
- Format:
- Link:
- License:
- Notes:
```

---

## How to Add a Paper

Please use this format:

```text
- Paper Title
  - Authors:
  - Venue/Year:
  - Dataset:
  - Task:
  - Method:
  - Metrics:
  - Code:
  - Notes:
```

---

## How to Add a Code Repository

Please use this format:

```text
- Repository Name
  - Task:
  - Dataset:
  - Framework:
  - Link:
  - Notes:
```

---

## Pull Request Guidelines

1. Add resources under the correct section.
2. Use official links whenever possible.
3. Avoid duplicate entries.
4. Keep descriptions short and clear.
5. Include dataset, method, and code links when available.
6. Do not add low-quality or unrelated links.
7. Follow dataset licenses and ethical use requirements.

---

## Quality Criteria

A resource is more likely to be accepted if it is:

- Publicly accessible
- Related to breast cancer AI research
- Useful for researchers or students
- Reproducible or well documented
- Published in a reputable venue or hosted by a reliable institution
